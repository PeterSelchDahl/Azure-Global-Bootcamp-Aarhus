GLOBAL AZURE BOOTCAMP 2018

Tap into the potential of the Microsoft Public Cloud

Sign up today!
Limited tickets available!

Global Azure BootCamp is a free one-day training event, taking place on the 21st of April 2018 in several venues worldwide, driven by local Microsoft Azure enthusiasts and experts.Our goal is to provide the catalyst for Azure communities to grow around the world, not just for this one day, but year round.

Target Audience

Developers, IT architects and IT-pros with an interest in knowing and learning more about the Microsoft Cloud Computing platform.

Networking

Get to know people, grow your network, exchange of valuable information on solutions in the Cloud, or simply chat about cloud technology.

Start your cloud journey

A full immersion day that will make you fit to develop cloud applications and create infrastructure in Microsoft Azure.

Keep secrets out of code with Managed Service Identity

12:15 - 13:00

Keep secrets out of code with Managed Service Identity.

Azure Active Directory MSI gives the runtime of your code an automatically managed identity for authenticating to Azure services, which allows you to keep credentials out of your code and separate concerns. In this session you will get an introduction to the concept and a demo of the functionality. Key technologies include Azure Active Directory, Web Sites, Functions, Key Vault and Azure SQL.

Going password less with Azure AD.

16:10 - 17:00

Going password less with Azure AD.

This session will give an introduction to the newly introduced modern authentication (FIDO / WebAuthn) option in Azure Active Directory and Windows 10 Redstone 4, which provides yet a not step in the right direction to eliminate passwords entirely. Obviously we all hate passwords and we all know that passwords are incredibly insecure because they’re based on shared secrets. But how can we run our business without any password? Come an hear about these amazing new possibilities and how we make the world more secure.

Windows Hello and FIDO2 Security Keys enable secure and easy authentication for shared devices.
